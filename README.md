# JSON Server Multiple Files/Resources

JSON Server: https://github.com/MaxGalliot/rest_api_recipes

### Install Global

```
npm install -g json-server
```

### Install local

```
npm install
```

## Structure

```
├── README.md
├── fakeapis
│   ├── index.js
│   ├── db.json
│   └── dataset_nutrition.json
├── package-lock.json
├── package.json
```

## Run JSON server

- `npm start`: Start json server multiple files
